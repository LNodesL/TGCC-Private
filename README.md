# TG Command Center

An easy to use CLI menu manager & bot service for Telegram.

Quickly modify commands.json file to create main menu options & sub-menus, for unlimited layers of menu triggers. 

Your bot runs CLI commands on your server(s) and returns results in code blocks via message.

Supports all TG chat types.

GUI or Text Menus, configurable in ENV.
